# Profile & Progress

Profile validated in sketch 003 (winner **A · Essentials**); the Progress tab was designed in
sketch 001. Sources: `sources/003-profile.html`, `sources/001-navigation-shell.html`.

## Profile: A · Essentials

Because **Progress owns its own tab**, the profile does NOT repeat metrics. It stays a calm,
low-stakes surface:

1. **Identity** — avatar (monogram), display name (Fraunces), "Learning English".
2. **Coach card** — the assigned coach (avatar + ink presence dot, "Your English coach", chevron). Assigned, never chosen. C · Relationship's warmer coach-card-up-front variant is available to graft on if a more personal feel is wanted.
3. **Settings** — Appearance (flips theme), Notifications, Language (English — the audience are non-native readers), Help & guide, Sign out. Each row ≥56px.

No primary button (a settings/identity surface). Rejected **B · Reflection** — a "what you've
worked on" block duplicates the Progress tab and edges toward a scoreboard.

## Progress tab: a milestone journey, never a scoreboard

- Vertical **milestone path**: done (`✓` filled `--ink` dot) → now (`●` ring) → up-next (hollow, muted). "James adds each step as you're ready."
- The current step carries a **visual bar** + **one calm phrase** ("A little more to go this week"). 
- **Progress is visual, never a grade.** No percentages-as-judgement, no ranks, no fractions pinned like a HUD.

## Reward-only, never a streak (hard rule)

A resettable day-count ("7 days practiced") is a **streak** — the #1 abandonment trigger for this
audience (AGENTS.md rule 5). It was removed. Any reward must be **non-decrementing** and have **no
failure state**: a calm phrase, cumulative-only counts, or milestone completion. **Returning after a
gap shows encouragement, never a lowered number.**

## Anti-patterns

- ❌ Stats/scoreboard on the profile (they live in Progress). ❌ Any streak that resets or freezes.
- ❌ A `3 of 5` fraction beside a bar with metas pinned to opposite edges (game-HUD grammar).
- ❌ Percentages, scores, ranks, or grades anywhere client-facing.
