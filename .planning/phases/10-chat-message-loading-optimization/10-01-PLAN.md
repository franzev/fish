from Phase 9. This plan is the shared contract every later plan consumes: new events, a new

finished contract instead of discovering it.

existing cases), updated protocol docs, and the phase-10 native notes.

New portable types this plan introduces (consumed by Plans 02/03 via structural typing):

```
type ChatMessageCursor = { createdAt: string; id: string };
interface ChatPaginationState {
  oldestLoadedCursor: ChatMessageCursor | null;
  hasMoreOlder: boolean;
  isLoadingOlder: boolean;
}
```

`mergeChatMessage(current, incoming, localRequestId?)` reconciles by `id`, `clientRequestId`, or
`localRequestId` then re-sorts by `compareChatMessages`. This already satisfies CLOAD-05 — the new
`olderPageLoaded` case loops it; do not write a second dedup function.
