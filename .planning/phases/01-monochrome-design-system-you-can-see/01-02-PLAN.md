Current Input contract (apps/web/components/ui/input.tsx) — harden in place:
- `interface InputProps extends InputHTMLAttributes<HTMLInputElement> { label: string; hint?: string; notice?: string }`
- Uses `useId()` + `inputId = id ?? autoId`, `forwardRef`, `Input.displayName = "Input"`
- Input classes: `w-full rounded-control bg-surface px-4 min-h-[var(--size-control)] text-[17px] text-foreground border border-border placeholder:text-muted transition-colors focus:border-primary`, `notice && "border-notice"`
- Message blocks: `{hint && !notice && <p className="mt-2 text-[14px] text-muted">...}` and `{notice && <p className="mt-2 text-[14px] text-notice">...}`
- IMPORTANT: the old `text-notice`/`border-notice` referenced an amber hue token removed in Plan 01. The token is now `--color-notice` (monochrome). `border-error`/`text-error` are the new tokens from Plan 01.

Current Card + Progress (apps/web/components/ui/card.tsx) — harden in place:
- `export function Card({ className, ...props }: HTMLAttributes<HTMLDivElement>)` — plain function (NOT forwardRef — Card is not focusable), classes `rounded-card bg-surface p-6 border border-border`
- `interface ProgressProps extends HTMLAttributes<HTMLDivElement> { value: number; label?: string }`
- Progress clamps 0–100, `role="progressbar"` + `aria-valuenow/min/max`, fill `bg-primary transition-[width] duration-500`

Two-tier signal rules (D-08/D-09), monochrome only:
- notice tier: `IconInfoCircle` (stroke 1.75, size 20), `border-border-strong` on the field, regular-weight message in `text-notice`
- error tier: `IconAlertCircle` (stroke 2, size 20), `border-error border-2` on the field, semibold (font-semibold) message in `text-error`
- Distinguishing signal is border weight + message weight + icon shape ONLY — never a `text-red-*` hue

Elevation (D-06): add `--shadow-card` to globals.css resolving to a soft shadow in light and `none` in dark via `light-dark()`. Card consumes it via a semantic utility (e.g. `shadow-[var(--shadow-card)]` or a `shadow-card` theme mapping) — NEVER `dark:shadow-none` (Pitfall 1, no dark: variant).

Icons (D-08/D-09): `import { IconInfoCircle, IconAlertCircle } from "@tabler/icons-react"`, `aria-hidden="true"` (adjacent copy carries the meaning).
