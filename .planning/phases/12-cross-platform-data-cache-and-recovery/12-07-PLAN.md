<threat_model>
## Trust boundaries

| Boundary | Untrusted input crossing it | Required control |
|---|---|---|
| Verified auth owner → Core Data fetch | Owner and conversation selectors | Exact predicates; missing or mismatched owner returns no snapshot |
| Accepted reducer result → private context | Metadata rows, tombstones, cursors, history truth | Whole-batch ownership validation and one transaction/save |
| Core Data store → ChatCore/PersonalChat | Stale, restored, or corrupted local rows | Safe DTO decode, verified owner gate, no cached authority |
| Store directory → iCloud/device migration | Private SQLite/WAL/SHM files | Complete file protection and backup exclusion |
| Cache limits/time → eviction | Clock values and row pressure | Injected clock, fixed tier order, protected newest 40 |

## STRIDE threat register

| Threat ID | Category | Component | Disposition | Mitigation and verification |
|---|---|---|---|---|
| T-12-01 | Information disclosure | Private cached metadata | mitigate | Compound owner/conversation constraints, verified-owner fetches, wrong-owner tests. |
| T-12-02 | Information disclosure | Memory-only delivery URLs | mitigate | Core Data model excludes URL leases/tokens/runtime references; model and SQLite sentinel scans. |
| T-12-03 | Spoofing / elevation | Owner isolation | mitigate | Local rows never authorize; exact owner/conversation predicates; cached capabilities omitted. |
| T-12-04 | Tampering / information disclosure | Identity transition | mitigate | Blocking `purgeOwner`, `verifyOwnerPurged`, and `sweepNonCurrentOwners` primitives consumed by Plan 06. |
| T-12-05 | Tampering / information disclosure | Filesystem/temp and backup | mitigate | Protected SQLite store, standardized store URL, backup exclusion, no raw deletion paths in the port. |
| T-12-06 | Information disclosure / repudiation | Logs/diagnostics | mitigate | Typed failures only; adapter never exposes provider strings, row dumps, URLs, or paths. |
| T-12-07 | Tampering | Stale callbacks | mitigate | Only Plan 01 accepted owner/generation/request results call transactions; repeated/stale writes are idempotent. |
| T-12-08 | Denial of service | Cache growth | mitigate | 400/conversation, 2,000/account, 30-day inactivity, and protected-newest tests. |
</threat_model>
