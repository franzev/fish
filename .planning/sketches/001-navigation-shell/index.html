FISH · sketch 001 · navigation
Design question

For an app whose whole job is to remove choices, what is the navigation?

Three structural models, each shown on web and Android together. Shared design language (tokens, Lexend / Fraunces, ≥56px targets), native structure on each platform. Toggle light / dark, bottom-right.

D · Synthesis — bottom nav · Home · Progress · Messages · Profile

Your direction: a familiar Messenger / WhatsApp shell. Three fixed destinations; Messages is a conversation list that opens into a thread. Web uses the desktop two-pane (list + open thread) with a left rail; Android uses a Material bottom nav. Home is the calm dashboard; Profile is the client. Tap the nav — and the coach row — it’s live.
